# a simplified version of MProcs
Files here are for documentation purposes and research. The distributable package 'mdcci' is installed as a single module from pip/pipx directory directly.

to install (distributable package = mdcci, enters lpro-s and has option to switch to lpro):

    pipx install mdcci

open:

    mdcci (default)
    lpro (included)
    lpro-s (included)
    lxpro (included)

to open and set user, Nano EST Charset, and display version/intro:

    lpro --user Nishes --charset korean --version

view Shell flags:

    mdcci --help

do indicated function then return to shell:
    
    lpro -biology --x

*lpro-s is a version which does not save to a .txt file
